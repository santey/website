S3Manager.

UploadCompressed(FilePath, string, UploadSettings, CancellationToken) Method

This content is part of a third party addin that is not supported by the Cake project. For more information about this project, please visit https://github.com/SharpeRAD/Cake.AWS.S3.

Summary

Uploads the specified file. For large uploads, the file will be divided and uploaded in parts using Amazon S3's multipart API. The parts will be reassembled as one object in Amazon S3.
