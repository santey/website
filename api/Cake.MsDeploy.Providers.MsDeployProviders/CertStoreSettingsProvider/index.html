CertStoreSettingsProvider Class

This content is part of a third party addin that is not supported by the Cake project. For more information about this project, please visit https://github.com/cake-contrib/Cake.MsDeploy.

Summary

This provider is used to sync Centralized SSL Certificates Support store settings. It does not require any path to be specified and can only be utilized on IIS 8 and later.
